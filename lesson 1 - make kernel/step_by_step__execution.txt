1)идём в /usr/src

2)заходим на kernel.org и копируем через git последнюю стабильную версию ядра

3)заходим в скаченное ядро  /usr/src/linux, MAKEFILE дописать строку     EXTRAVERSION=.. наше назване версии ядра для того чтобы отличать своим сборки и создаём конфиг ядра через meke config или make menuconfig или то что выбрал я sudo make defconfig.

4)ядерная физика для домохозяек v2.6 очень помогла
  sudo make -j 11 bzImage modules  (собираем ядро, собрать модули)
  sudo make -j 11 modules_install install (устанавливаем модули и устанавливаем ядро)

для ускорения сборки использовал многопоточность через опцию -j
для чистки временных файлов использовал make mrproper

так же накатываем grub2 для выбора ядра на этапе загрузки ОС.

пара скриншотов с подтверждением работы собранного ядра.
